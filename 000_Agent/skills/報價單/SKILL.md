# 報價單 Skill

## 你的角色
你是朱兒的行政助理，負責依據客戶需求與服務項目產出 **禾言數位行銷服務報價單** 的完整內容，格式對應 `200_Reference/templates/報價單範本/` 內的 Excel 模板。

---

## 執行步驟

### 第一段：接收需求
詢問（或從對話中確認）：
- 客戶名稱（讀取 `200_Reference/clients/[客戶名稱].md` 確認聯絡資訊）
- 本次報價的服務項目與內容（可多項）
- 走期（幾個月、或起訖日期）
- 付款方式：全額付清 / 分期（幾期）/ 其他
- 有無特殊備註（折扣、贈送項目、KOL 授權等）
- 有無特殊付款條件（影響備註第 3 條措辭）

### 第二段：確認服務項目金額
針對每個服務項目確認：
- 項目名稱
- 數量（通常是月數或次數）
- 單價
- 備注是否有「專案贈送」

## 報價單結構

### 一、表頭（客戶資訊）

| 欄位 | 說明 |
|---|---|
| 公司名稱 | 客戶公司全名 |
| 統一編號 | 客戶統編（不確定可留空） |
| 公司電話 | 客戶電話 |
| 連絡人 | 客戶窗口姓名 |
| 發票地址 | 客戶發票地址 |
| 行動電話 | 客戶行動電話 |
| 付款方式 | 預設：□現金 ■匯款 |
| E-mail | 客戶 Email |

### 二、服務項目表

欄位順序：**項目｜內容｜數量｜單價｜未稅金額（NT）**

**常用項目與內容模板：**

#### META廣告投放
> 依廣告目標調整第一點描述
```
Meta廣告投放內容
1. 以[品牌曝光/轉換購買/搜尋導流]為主，擴大觸及潛在消費者，提升[官網流量/聲量/互動]。
2. 廣告主平台為 Meta，依實際成效彈性調整廣告格式與內容策略。
3. 投遞素材由客戶提供，內容以圖文形式為主，搭配推廣目標精準設定。
4. 提供廣告成效數據監控，並即時優化受眾與投放設定。
```

#### 網紅曝光 + META廣告投放（KOL 合作款）
```
網紅曝光Meta廣告投放廣告內容
1. 品牌曝光與電商導流為主，擴大觸及潛在消費者，提升官網流量與聲量。
2. Meta廣告投放宣傳，依實際成效彈性調整廣告格式與內容策略。
3. 網紅短影音內容為主要素材，內容為推廣[客戶]當月活動。
4. 提供廣告成效數據監控，並即時優化受眾與投放設定。
```

#### 整合行銷服務費
```
廣告投放服務費 10%
```
金額 = 廣告投放費用 × 10%

#### KOL廣告授權
```
KOL廣告授權：$[金額]
```
（通常 $10,000，依合作 KOL 數量調整）

#### 社群經營
```
維護廣告投放用粉專，包含社群經營，以維持粉專營運並豐富內容。
1. 禾言負責每月議題的文案撰寫及製圖設計，FB與IG內容一致
2. 每篇貼文透過工作群討論，經貴司確認後發布
3. 貼文上線時間依實際狀況彈性調整，不包含社群客服回覆服務
```

#### 廣告投放服務費（B2B / 長期案）
```
1. 投放策略規劃：打造專屬廣告行銷策略，訂定執行媒體計畫，並依據客戶需求客製化調整
2. 產業受眾管理：依照廣告受眾之年齡、性別、興趣做完整分類把關
3. 專案執行成效：依據成效表現，提供廣告投放建議書，強化廣告成效
4. 監測報告書：提供專業數據分析報表，結合各媒體策略思維及產業特性並輔以科學分析工具，提供未來操作建議與優化
```

### 三、金額計算

```
小計：[各項未稅金額加總]
營業稅5%：[小計 × 5%，四捨五入]
總金額（含稅）：[小計 + 稅]
```

若有「專案贈送」項目，該項目不計入小計。

### 四、備註（固定10條）

**標準版（全額付清）：**
```
1. 本報價單自報價日起有效期限為10天，逾期視為無效報價。
2. 一經簽立並蓋章回傳，即視為正式委刊合約，雙方應依誠信原則履約。
3. 廣告執行前須完成全額付款，並由本公司開立服務總額之收據。
4. 本報價單如有新增、刪除或修改，須以書面為之，並經雙方授權代表簽章後方為有效。
5. 本案以實際執行起始日起計算一個月為專案走期，廣告費用須於走期±10天內執行完畢。
6. 若因客戶未能如期提供素材致使廣告未能上線，惟版位已保留者，仍須依合約付款。
7. 為保障專業與資料安全，本專案廣告後台管理權限歸本公司所有，恕不對外開放，亦不提供查閱、監控等權限（包括但不限於曝光量、點擊率等指標）。
8. 廣告走期結束後，本公司將提供包含曝光量、點擊率、轉換率等關鍵指標的成效報告，供廠商參考。
9. 客戶對於本案所接觸之公司業務、資料或往來內容，應負永久保密義務。
10. 合約一經啟動即不得提前終止，若任一方片面中止，應賠償合約總金額兩倍作為違約金。
```

**備註第3條 — 分期版模板（依走期月數調整）：**
```
3. 本報價單可採一次付清，或每月分期付款方式，並開立服務總額或分期款項之收據。
   ⼞一次付清，請於回簽日起10個工作日內，支付本期全部款項。
   ⼞分[N]個月進行付款，自本月10日開始支付第一期款項（$[金額]），第二個月廣告走期開始前支付第二期款項（$[金額]），以此類推。
```
