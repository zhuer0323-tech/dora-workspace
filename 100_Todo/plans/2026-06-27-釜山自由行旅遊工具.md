# 釜山自由行旅遊工具

> 建立時間：2026-06-27
> 類型：新功能
> 狀態：待執行

---

## 你要做的事（一句話版）

做一個可以加到手機桌面、像 App 一樣使用的釜山旅遊小工具，整合行程、地圖、匯率、消費記錄、天氣穿搭五大功能。

## 背景和動機

六天五夜的釜山自由行，不想帶著一堆工具東查西查。希望一個工具搞定：看行程、開地圖、換算花費、記帳、查天氣決定穿什麼。

## 具體步驟

### Step 1：搭建基本骨架（外觀 + 頁面切換）
- 做什麼：建一個有底部導覽列的網頁，五個分頁分別對應五大功能
- 產出：可以在手機瀏覽器打開、點底部 icon 切換頁面的基本框架
- 注意：設計要適合手機螢幕，字要夠大、按鈕要夠好點

### Step 2：行程頁面
- 做什麼：依照六天五夜的格式，每天列出地點與活動，每個地點有「開地圖」按鈕連結到 Naver Map；另有「新增/刪除」功能讓你自己調整
- 產出：可以看行程、可以編輯行程、點按鈕直接跳去 Naver Map 導航
- 注意：需要你提供出發日期和初版行程內容（也可以做完工具再填）

### Step 3：即時匯率顯示
- 做什麼：從免費的匯率查詢服務即時抓取台幣兌韓元匯率，顯示在網頁上，並提供快速換算輸入框（輸入台幣 → 顯示韓元，或反過來）
- 產出：一個小型換算工具，出發前和在韓國都能用
- 注意：需要有網路才能抓最新匯率；若免費 API 有每日限制，會自動快取最後一次的匯率備用

### Step 4：消費記錄
- 做什麼：輸入韓元金額和消費分類（餐飲/交通/購物/住宿/其他），系統根據即時匯率自動顯示台幣換算結果，並累計每天和整趟旅程的總花費
- 產出：旅途中可以即時記帳、回國後能看到整趟共花了多少台幣
- 注意：記錄會存在手機瀏覽器裡，換瀏覽器或清除快取會消失；如需備份可考慮匯出功能（視時間決定是否加）

### Step 5：天氣 + 穿搭建議
- 做什麼：自動抓取釜山即時天氣（氣溫、降雨、風速），根據天氣狀況給出穿搭建議，例如「今天 18°C 有風，建議穿薄外套 + 長褲」
- 產出：打開工具就能看到釜山今天的天氣，以及今天出門該穿什麼
- 注意：使用免費天氣 API，需要有網路；穿搭建議依氣溫區間設定規則（不用 AI 即時生成，速度更快）

### Step 6：讓它可以加到手機桌面
- 做什麼：加入必要的設定，讓手機瀏覽器在打開網頁時，顯示「加到主畫面」的提示；加入後就像一個 App 一樣有自己的圖示，全螢幕顯示
- 產出：手機桌面有一個「釜山旅遊」的圖示，點開直接進入工具
- 注意：iOS（iPhone）和 Android 的安裝步驟略有不同，我會在做完後附上說明

### Step 7：上傳到可以隨時開啟的地方
- 做什麼：把做好的工具放到 GitHub Pages 或類似的免費網頁空間，取得一個網址，手機電腦都能開
- 產出：一個固定網址，隨時可以從瀏覽器打開或分享給旅伴
- 注意：和你現有的數學遊戲、Threads 工具放在同一個地方

## 預計成果

做完以後你會得到：
- 一個釜山旅遊專屬的網頁工具，可以加到手機桌面像 App 一樣使用
- 六天五夜行程一覽，每個地點一鍵開 Naver Map 導航
- 即時台幣 ↔ 韓元換算，在韓國消費時直接查
- 記帳功能，輸入韓元自動換算台幣，回國看總花費
- 釜山天氣 + 今日穿搭建議，出門前快速查一下

## 不包含在這次的範圍

- 行程的旅館/機票/景點預訂功能（只看行程，不做訂位）
- 離線地圖（地圖導航需要開啟 Naver Map，需有網路）
- 消費記錄雲端同步（記錄只存在你的手機裡）
- AI 即時生成穿搭建議（用預設規則，不呼叫 AI）

## 可能遇到的風險

- **免費 API 額度不夠**：天氣和匯率用的是免費服務，每天有查詢次數限制。處理方式：加快取機制，同一個數據每小時只抓一次，不會超額
- **iPhone 加到主畫面的限制**：iOS 的「安裝」體驗比 Android 差一點，沒有自動提示。處理方式：做完後我會附上 iPhone 的安裝步驟截圖說明
- **在韓國換瀏覽器**：如果臨時換了瀏覽器，之前的消費記錄不會跟著跑。處理方式：可以加一個「匯出為文字」的功能，讓你備份記錄
