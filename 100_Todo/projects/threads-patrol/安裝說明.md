# Threads 海巡工具 — 安裝說明

## 需要準備的東西
- Mac 電腦
- Threads 帳號

---

## 安裝步驟（只做一次）

### 第一步：安裝 Node.js
打開 Terminal，貼上這段指令後按 Enter：

```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
```

關掉 Terminal，重新開，再貼：

```bash
nvm install 20
```

### 第二步：下載工具資料夾
把整個 `threads-patrol` 資料夾複製到你的電腦（放在桌面或 Downloads 都可以）

### 第三步：安裝套件（只做一次）
打開 Terminal，輸入：

```bash
cd ~/Desktop/threads-patrol   # 改成你放的位置
npm install
npx playwright install chromium
```

### 第四步：填入你的 Groq API Key
用記事本打開 `.env` 檔案，填入你的 API Key：

```
GROQ_API_KEY=你的key
```

> Groq 免費申請：https://console.groq.com

---

## 每次使用

1. 雙擊 `啟動海巡工具.command`
2. Terminal 自動開啟，瀏覽器跳出來
3. **第一次使用**：瀏覽器會開到 Threads，先手動登入你的帳號，登入後工具會自動記住
4. 之後就直接在網頁操作

---

## 常見問題

**問：雙擊 .command 跳出「無法開啟」？**
在 Terminal 輸入：
```bash
chmod +x ~/Desktop/threads-patrol/啟動海巡工具.command
```

**問：忘記 Groq API Key 怎麼辦？**
去 https://console.groq.com 重新申請，免費的。

**問：搜尋一直失敗？**
關掉工具，重新雙擊啟動，等瀏覽器完全開啟再試。
