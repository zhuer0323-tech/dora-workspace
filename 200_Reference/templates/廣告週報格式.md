# 廣告週報格式（LINE 訊息風格）

## 格式範例
```
🔸[素材名稱]
▶️累積曝光數約 X 萬次
▶️ROAS達 X.X

🔸[素材名稱]
▶️累積曝光數約 X,XXX 次
▶️ROAS達 X.X

🔺觀察資訊發現[分析重點]，[說明原因]；[第二個觀察]，[說明原因]。
🔺[針對特殊情況的補充說明，例如剛上線廣告的觀察提醒]
```

## 規則
- 每則廣告用 🔸 開頭，後接素材名稱
- 每個指標用 ▶️ 開頭
- 結尾用 🔺 做觀察分析（1-3 條）
- 語氣親切、口語，最後一句可加「唷!」等軟化語氣
- 指標格式：曝光超過 1 萬用「約 X.X 萬次」，未達 1 萬直接寫數字加逗號
- 核心指標：曝光數、ROAS（若有購買轉換）；流量廣告則報曝光數與點擊數

## 分析寫法重點
- 點出表現最好的素材與原因（受眾洞察、素材角度）
- 點出表現次佳的素材
- 若有剛上線素材，說明觀察期提醒
- 不評論表現差的廣告（除非有特別原因）
