# 朱兒廣告文案語氣風格分析

## 整體風格
活潑親切、結構清晰、善用 emoji 輔助視覺分層。
不過度煽情，但會適度製造緊迫感與期待感。
B2C 偏感性訴求，B2B 偏利益點列舉。

---

## 開頭 Hook 常用模式

**1. 痛點問句**
```
「人海茫茫，該怎麼找到你的顧客❓」
想打造最美造景❓傳統設計無法滿足需求❓
擔心戶外蚊蟲侵害❓造景設計遭害蟲破壞❓
```

**2. 衝突陳述**
```
『錢都砸了，卻還是沒！訂！單❗❓』
氣溫下降，容易手腳冰冷❓
```

**3. 節日 / 情境帶入**
```
📢⚟寵愛媽咪趁現在！滿額獻禮
🎉 好運龍厚禮 🎁 新春送禮首選
```

---

## 結構框架

**B2C 電商類**
```
節日/情境 Hook
↓ 產品特色（3-4點列舉）
↓ 社會證明（年份、獲獎、數字）
↓ 促銷資訊（限時、滿額贈）
↓ CTA + 連結
↓ Hashtag（8-15個）
```

**B2B / 服務類**
```
痛點問句
↓ 解決方案一句話
↓ 4大優勢列舉
↓ 免費諮詢 CTA
↓ 聯絡方式
↓ Hashtag（5-8個）
```

---

## Emoji 使用慣例

| 用途 | 常用符號 |
|---|---|
| 區塊標題 | 📢⚟、❚、▍ |
| 列點 | ✅ ✔️ ✦ ➠ ▶️ |
| 強調 | 🎖️ ✨ 🔥 💪 |
| CTA | ➥ ▶ 👉 ⇢ ↪︎ |
| 分隔 | `\\ 內容 //`、`❯❯` |
| 數字列表 | 1️⃣2️⃣3️⃣ |

---

## 語氣關鍵字

**製造行動感**：立即、趁現在、限時、只到、搶

**增加信任感**：40年老字號、政府補助、獲獎認證、維他命C是檸檬的30倍

**促銷語氣**：滿額贈、9折、88折、免費、加贈

**CTA 慣用語**：
- 立即私訊、點擊連結、立即下單
- 專人為您諮詢、免費規劃
- 立即詢問 + 服務項目

---

## 產品類別語氣差異

| 類型 | 語氣特色 |
|---|---|
| 食品 / 電商 | 溫暖、食慾感強、強調天然真材實料 |
| 健康 / 養生 | 數據佐證（X倍）、功效列點清楚 |
| B2B 廣告服務 | 強悍有力、直指痛點、強調「快狠準」 |
| 課程培訓 | 機會感、政府加持、年輕族群語氣 |
| 開幕促銷 | 喜慶、大方、好禮條列清楚 |

---

## 撰文前的 Checklist

- [ ] 開頭有沒有 Hook（問句 or 情境）
- [ ] 產品特色列舉 3-4 點，不超過 5 點
- [ ] 有沒有數字或社會證明
- [ ] 促銷資訊清楚標示
- [ ] CTA 明確（動詞開頭）
- [ ] Hashtag 精準相關（不超過 15 個）
- [ ] 全文讀起來有沒有節奏感（不要一整塊文字）
